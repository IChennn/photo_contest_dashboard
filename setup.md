# Photo contest dashboard

Automatically scrapes Japanese photo contest aggregator sites daily
and serves the results as a static dashboard on GitHub Pages.

## Setup

### 1. Create a new GitHub repository and push

```bash
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/<your-username>/photo-contest-dashboard.git
git push -u origin main
```

### 2. Enable GitHub Pages

Settings → Pages → Source: **Deploy from a branch**
→ Branch: `main` / Folder: `/docs` → Save

### 3. Run the scraper for the first time

Actions tab → "Refresh photo contest" → Run workflow

After that it runs automatically every day at 09:00 JST.

---

## Adding a new source

Open `scraper/sources.py` and add one line to the `SOURCES` list:

```python
{ "name": "GANREF", "url": "https://ganref.jp/photo_contests/jpn/" },
```

That's it. Pagination is handled automatically.
If the site has unusual markup and the generic scraper doesn't pick anything up,
open `scraper/scrape.py` and add a custom parser there.

---

## Running locally

```bash
pip install -r scraper/requirements.txt
python scraper/scrape.py
# output → docs/data/contests.json
```

---

## How it works

```
Every day at 09:00 JST
        │
        ▼
GitHub Actions runs scraper/scrape.py
        │
        ├── reads source list from scraper/sources.py
        ├── fetches each site (auto-pagination)
        ├── parses contest name / deadline / prize
        └── writes docs/data/contests.json
                │
                ▼
        git commit & push
                │
                ▼
GitHub Pages serves docs/index.html
        │
        └── browser fetches data/contests.json on load
```

## File structure

```
photo-contest-dashboard/
│
├── .github/
│   └── workflows/
│       └── scrape.yml          # Cron job — daily 09:00 JST
│
├── scraper/
│   ├── sources.py              # ★ Edit this to add/remove sources
│   ├── scrape.py               # Scraper engine (generic + custom parsers)
│   └── requirements.txt        # Python dependencies
│
└── docs/                       # Served by GitHub Pages
    ├── index.html              # Dashboard UI
    └── data/
        └── contests.json       # Generated by scraper — do not edit manually
```
